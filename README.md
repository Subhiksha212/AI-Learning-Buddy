# 🎓 AI Learning Buddy

## 📖 Overview

AI Learning Buddy is an AI-powered educational web application developed using **Python**, **Streamlit**, and the **Google Gemini API**. The application helps students understand academic concepts through simple explanations, real-life examples, quizzes, and interactive conversations.

The application provides a ChatGPT-like interface where learners can ask questions on any topic and receive instant AI-generated responses.

---

## 🚀 Features

- 📚 Explain any academic concept in simple language
- 🌍 Provide real-life examples
- ❓ Answer questions on any topic
- 💬 Interactive chat interface
- 📝 Chat history with automatically generated titles
- ➕ Start a new chat session
- 🤖 Powered by Google Gemini AI
- 🎨 Clean and user-friendly interface

## ⚙ Installation

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Learning-Buddy.git
```

### 2. Open the project

```bash
cd AI-Learning-Buddy
```

### 3. Install the required libraries

```bash
pip install -r requirements.txt
```

---

## 🔑 Configure Gemini API Key

Create a **Secrets** file (or add Secrets in Streamlit Cloud):

```toml
GEMINI_API_KEY="YOUR_GEMINI_API_KEY"
```

Do **not** store your API key directly in `app.py`.

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open in your browser.
